# Number Guessing Game

## Description:
This is a terminal based number guessing game where the computer generates a random number and the user attempts to guess it. The user picks a difficulty level (Easy, Medium, or Hard) and then inputs a guess. The program provides feedback after each guess ("Too high" or "Too low") until the correct number is guessed.

## How To Run
1. Have python installed on your computer
2. Download or clone the repository from github
3. Open a terminal (CommanyPrompt, PowerShell, or MacTerminal)
4. Navigate to project folder using cd (cd/path)
5. Run the program using python GameCode.py (or python3 on Mac)
