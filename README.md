# Number Guessing Game

## Description:
The game is a terminal-based number guessing game that includes difficulty levels and the ability to replay. The computer randomly generates a number that the user must guess. The user first selects a difficulty level: Easy (1-50), Medium (1-100), or Hard (1-200). 

After each guess, the program provides feedback (“Too high” or “Too low”) until the correct number is guessed. Once correct, the game ends, displays the number of attempts taken, and asks the user if they wish to play again. 


## How To Run
1. Have python installed on your computer
2. Download or clone the repository from github
3. Open a terminal (CommandPrompt, PowerShell, or MacTerminal)
4. Navigate to project folder using cd (cd/path)
5. Run the program using python GameCode.py (or python3 on Mac)
